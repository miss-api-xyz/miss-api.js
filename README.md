## Install 
```shell
$ npm i miss-api.js
```
or u can use yarn
```shell
$ yarn add miss-api.js
```

## How to use?

```js
const { main: MissAgent } = require("miss-api.js"); //
const miss = new MissAgent(`User ...`) // API Key. 

miss.get("cat").then(console.log) // return object with data
```
